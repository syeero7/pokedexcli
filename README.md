# Pokedex CLI

This is a [REPL](https://en.wikipedia.org/wiki/Read%E2%80%93eval%E2%80%93print_loop) CLI application that utilizes data fetched from the [PokeAPI](https://pokeapi.co/). It allows users to look up detailed information about [Pok√©mon](https://en.wikipedia.org/wiki/Pok%C3%A9mon).

## Getting Started

Clone the project

```bash
  git clone https://github.com/syeero7/pokedexcli.git
```

Go to the project directory

```bash
  cd pokedexcli
```

Launch the CLI

```bash
  go run .
```

## Running Tests

To run tests, run the following command

```bash
  go test ./...
```
